**Implemented features**
- **Add note:** Create a note with a title and content.
- **Tags:** Add multiple tags to a note via the tag input and add button.
- **Render tags:** Tags are shown on each note and in the tag manager UI.
- **Delete note:** Delete notes with the trash button on each note.
- **Search:** Full-text search across titles, content, and tags.
- **Filter by tag:** Dropdown to filter notes by tag.

**Credits**

**How to run the project locally**

- **Option A — VS Code Live Server extension:** Install and click "Go Live".
- 


- **Option B — Recommended (local static server):**

```bash

npx http-server -c-1 .  # or: npx serve .

```

- **Option C — Quick (no server):** Open `index.html` in a browser by double-clicking the file. Some features may require a server for correct MIME handling.
